Map Title

Map authored by Lauren Winkler

This map shows the percentage of occupied housing units in Kentucky which were rented in 2010. These data were obtained through American Fact Finder; 2010 Census, SF1 and have been aggregated by county.

Renter-occupied housing percentages have been classified using the ckmeans statistical method. This means that counties with statistically similar rentership percentages are shown in the same color. All maps distort data in some way, but this classification method provides far less distortion than equal interval, quantile, or other manual classification methods.

This map shows that the state's urban counties (like Fayette, Hardin, and Jefferson counties) have the highest percentage of rented housing. (Note the similarities between rentership and population density). There are, however, a few outliers for the some of the least populated counties. Representing the data over areas of equal population, instead of counties, would eliminate this bias. Overlaying this data on a basemap showing Kentucky's cities and universities would also help support the map's story.
